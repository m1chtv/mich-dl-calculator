# mich-dl-calculator
A lightweight, browser-based **file downloader UI** with smart link handling and clean feedback.

## Features
- Direct download via URL
- Auto filename detection
- Progress-style visual feedback
- Error handling for invalid links
- Minimal, fast, distraction-free UI
- Fully responsive layout

## Tech Stack
- HTML5
- CSS3 (custom)
- Vanilla JavaScript

## Usage
Open `dl.html`, paste a direct file URL, and start downloading.  
No install. No backend. Works fully client-side.

## Notes
- Uses native browser download behavior
- No data is stored or tracked
- Designed for speed and simplicity
